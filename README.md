Algorithm
Read the input year.
Check if the year is divisible by 400.
If not, check if it is divisible by 100.
If not, check if it is divisible by 4.
Print "True" if it is a leap year; otherwise print "False".
Example

Input

1990

Output

False

Explanation
1990 is not divisible by 4, so it is not a leap year.

Time Complexity
O(1) – Constant time, as only a few arithmetic operations are performed.
Space Complexity
O(1) – Constant space, as no extra data structures are used.
